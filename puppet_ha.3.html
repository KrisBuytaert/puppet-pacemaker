NAME

puppet_ha - A Puppet module for complete management of the Linux-HA stack

INSTALL

Download and extract into your Puppet modules directory (eg `/etc/puppet/modules').

  • Tarball (latest)
  • Zipball (latest)

BUGS

  • Does not manage resource operations (eg "op monitor foo")

CONTRIBUTING

  • Fork the project on GitHub.
  • Make your feature addition or bug fix.
  • Add tests for it.
  • Commit, but do not mess with the Rakefile, version or history
  • Send me a pull request
